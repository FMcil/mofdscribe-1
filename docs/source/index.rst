mofdscribe |release|
==================================

mofdscribe is a Python library for featurizing metal-organic frameworks (MOFs).
nofdscribe contains 10+ featurizers that have been adapted from scientific publications and are accessible using the consistent and battle-proof `matminer <https://hackingmaterials.lbl.gov/matminer/>`_ API.


mofdscribe is open source via a MIT license.
